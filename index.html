Ruby on Trails
A Full Stack app built with Ruby on Rails. It uses Devise for user authentication, Ospry API for image upload, Google Maps API and geocoding to place location markers on the map, and Disqus for users to leave comments. The database is managed with PostgreSQL.

Sound Board
